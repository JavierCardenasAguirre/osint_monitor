Proyecto creado con el fin de hacer seguimiento a las noticias de medios locales y grupos de facebook 
donde publican noticias de interes de los departamento de Córdoba y Antioquia

se proyecta mapa con interaccion tipo similar kaspersky ciberamenazas

se añaden tipologias

diccionario de palabras clave para identificar las noticias y clasificarlas

proyecto creado con fines educativos con expansion a mas departamentos de Colombia

base de datos = Supabase

uso de vercel como alojamiento para hacer pruebas de testeo, pero se puede usar cualquier dominio de pago "economico" que prefieras
