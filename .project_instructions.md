# OSINT Monitor - Córdoba & Antioquia

## Overview
Plataforma de monitoreo OSINT (Open Source Intelligence) para noticias de los departamentos de Córdoba y Antioquia, Colombia. Dark theme, responsive.

## Technical Stack
- App Router with Server/Client components
- PostgreSQL via Prisma ORM (tabla "noticias")
- MapLibre GL for interactive map
- Recharts for radar chart
- Framer Motion for animations
- TailwindCSS dark theme

## Visual Style
- **Theme**: Dark mode by default (hsl 224 20% 6% background)
- **Primary color**: Green (#16a34a / hsl 152 80% 45%)
- **Font**: DM Sans (body), Plus Jakarta Sans (display), JetBrains Mono (data)
- **Cards**: bg-card with border-border/50, rounded-xl, shadow-md

## Key Features
1. Interactive map with color-coded markers by tipología
2. Filters: departamento, municipio, date range, tipologías, text search
3. News cards grid (1 col mobile, 4 col desktop)
4. Radar chart of tipologías
5. 7-day timeline
6. Add news modal with auto-classification (DICCIONARIO_INTEL)
7. Total news counter

## Data
- 16 tipologías with color coding (COLORES map)
- 30 Córdoba + 125 Antioquia municipalities with coordinates
- Auto-classification via keyword matching in DICCIONARIO_INTEL

## API Routes
- GET /api/noticias - filterable list with pagination
- POST /api/noticias - create new noticia with auto-classification
- GET /api/noticias/stats - tipología counts + 7-day timeline
